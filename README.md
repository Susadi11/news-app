# news-app

Project Setup

1. Clone the repository
    ```bash
   git clone https://github.com/Susadi11/news-app.git

2. Navigate to the news-app folder
   ```bash
   cd news-app

3. Navigate to the frontend folder
   ```bash
   cd frontend

4. Install the dependencies
   ```bash
   npm install

5. Run the development server
   ```bash
   npm start
